# Skill Map

운영 하네스에서 사용하는 스킬 맵입니다.

## 스킬별 담당 범위

- `harness`
  - 요청 분류와 표준 순서 안내
- `eso-recurrence-prevention`
  - ESO, CRD, Secret 동기화 재발 방지
- `cluster-reprovision-handoff`
  - 재구축 이후 인프라 인계
- `k8s-release-verification`
  - 배포 후 검증과 롤아웃 문제 분류
- `infra-ops`
  - Terraform 중심 인프라 작업

## 관계

- `harness`가 상위 라우터 역할
- 나머지 스킬은 실제 작업 스킬 역할
