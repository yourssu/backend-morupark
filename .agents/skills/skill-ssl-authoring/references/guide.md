# skill-ssl-authoring/references 가이드

이 디렉토리는 스킬 작성과 변환에 필요한 상세 참고 문서를 모아둔 곳입니다.
상위 `SKILL.md`가 짧은 실무 안내라면, 이 폴더는 작성 기준과 절차를 자세히 설명하는 역할을 합니다.

## 파일 설명

- `conversion-playbook.md`
  - 기존 텍스트형 스킬을 `SKILL.md + ssl.json` 구조로 바꾸는 절차를 설명합니다.
  - 어떤 트리거를 뽑아야 하는지, scene을 어떻게 나누는지, grounding을 어떻게 붙이는지 단계별로 안내합니다.
- `ssl-schema-guide.md`
  - `ssl.json`의 권장 스키마를 설명합니다.
  - 어떤 최상위 키가 필요한지, 각 scene/step이 어떤 필드를 가져야 하는지, 검증 시 무엇을 확인해야 하는지 정리합니다.

## 어떻게 사용하면 되나

- 기존 문서를 구조화된 스킬로 변환할 때는 `conversion-playbook.md`를 먼저 봅니다.
- `ssl.json`의 필드 구조나 검증 기준이 헷갈릴 때는 `ssl-schema-guide.md`를 확인합니다.
- 실제 작성은 상위 디렉토리의 템플릿과 함께 보는 것이 가장 효율적입니다.

## 수정할 때 팁

- 이 폴더의 문서는 "원칙"과 "패턴"을 설명하는 데 집중하는 것이 좋습니다.
- 특정 저장소 상황에만 맞는 예시는 너무 많이 넣지 말고, 필요하면 상위 스킬 문서로 올려서 분리하는 편이 좋습니다.
