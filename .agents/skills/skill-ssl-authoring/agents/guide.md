# skill-ssl-authoring/agents 가이드

이 디렉토리는 `skill-ssl-authoring` 스킬의 표시용 메타데이터를 담는 곳입니다.
스킬의 실제 절차를 정의하는 디렉토리가 아니라, 이름과 설명을 노출하기 위한 보조 영역입니다.

## 파일 설명

- `openai.yaml`
  - 스킬의 표시 이름, 짧은 설명, 기본 호출 문구를 정의합니다.

## 어떻게 사용하면 되나

- 스킬이 목록에서 어떻게 보일지 바꾸고 싶을 때 이 파일을 수정합니다.
- 스킬 작성 규칙이나 단계 흐름을 바꾸고 싶다면 상위의 `SKILL.md`, `ssl.json`, `references/`를 수정해야 합니다.
