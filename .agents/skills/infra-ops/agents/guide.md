# infra-ops/agents 가이드

이 디렉토리는 `infra-ops` 스킬의 에이전트 표시용 메타데이터를 담습니다.
실제 실행 절차를 정의하는 곳은 아니고, 스킬을 어떤 이름과 설명으로 노출할지 정리하는 용도입니다.

## 파일 설명

- `openai.yaml`
  - 스킬의 표시 이름(`display_name`), 짧은 설명(`short_description`), 기본 호출 문구(`default_prompt`)를 정의합니다.

## 어떻게 사용하면 되나

- 스킬 이름이나 설명을 더 이해하기 쉽게 바꾸고 싶을 때 이 파일을 수정합니다.
- 스킬의 실제 동작 순서를 바꾸려면 여기 말고 상위 디렉토리의 `SKILL.md`나 `ssl.json`을 수정해야 합니다.

## 수정할 때 팁

- 여기에는 UI용 짧은 문장만 두는 것이 좋습니다.
- 긴 절차 설명이나 규칙은 이 디렉토리가 아니라 상위 문서에 두는 편이 관리하기 쉽습니다.
