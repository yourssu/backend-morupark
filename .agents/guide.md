# .agents 가이드

이 디렉토리는 이 저장소에서 사용하는 에이전트용 보조 문서와 스킬을 모아두는 루트입니다.
코드 실행 로직 자체보다는 "에이전트가 어떤 문서를 읽고 어떤 절차로 작업해야 하는지"를 정의하는 메타 레이어에 가깝습니다.

## 이 디렉토리의 역할

- 저장소 전용 에이전트 설정과 스킬을 한곳에 정리합니다.
- 반복적으로 수행하는 작업을 문서화해서 재사용할 수 있게 합니다.
- 사람용 설명(`SKILL.md`)과 기계가 읽기 쉬운 구조화 문서(`ssl.json`)를 함께 관리합니다.

## 포함된 하위 디렉토리

- `skills/`
  - 실제 스킬 패키지들이 들어 있습니다.
  - 각 스킬은 보통 하나의 독립 디렉토리로 관리됩니다.

## 어떻게 사용하면 되나

1. 먼저 `skills/` 아래에서 필요한 스킬을 찾습니다.
2. 각 스킬 디렉토리의 `SKILL.md`를 읽어 사람이 이해할 수 있는 개요를 파악합니다.
3. 더 정밀한 구조가 필요하면 같은 디렉토리의 `ssl.json`을 확인합니다.
4. 세부 절차나 예시는 `references/`, 템플릿은 `assets/`, UI 메타데이터는 `agents/`에서 찾습니다.

## 수정할 때 팁

- 새 스킬을 추가할 때는 `skills/` 아래에 새 디렉토리를 만들고 기존 스킬 구조를 참고하면 됩니다.
- 사람이 읽는 설명과 기계가 읽는 구조를 섞지 말고 역할을 분리해서 유지하는 편이 좋습니다.
